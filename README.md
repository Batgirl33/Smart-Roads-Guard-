# Smart-Roads-Guard-

## Overview

Smart Roads Guard is a graduation project designed to improve road safety by detecting obstacles in real time using Computer Vision and Machine Learning. The system identifies different types of road obstacles and provides alerts to help reduce accidents and support safer driving.

---

## Objectives

* Detect road obstacles in real time.
* Improve road safety by providing early warnings.
* Classify different obstacle types using a trained deep learning model.
* Demonstrate the application of Artificial Intelligence in intelligent transportation systems.

---

## Dataset

The dataset was collected and prepared form many datasets.It was cleaned, organized, and annotated to train the YOLOv8 model for recognizing different types of road obstacles.

---

## Project Workflow

1. Collect and prepare the dataset.
2. Annotate and classify obstacle images.
3. Train the YOLOv8 model.
4. Evaluate the model's performance.
5. Detect obstacles in real-time images or videos.
6. Generate alerts when obstacles are detected to linked website 

---

## Features

* Real-time obstacle detection.
* Multiple obstacle classification.
* AI-based object detection using YOLOv8.
* Improved road safety through early warning.
